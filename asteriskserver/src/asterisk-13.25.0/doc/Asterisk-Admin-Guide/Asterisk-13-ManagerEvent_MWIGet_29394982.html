MWIGet

Synopsis

Raised in response to a MWIGet command.

Description

Syntax

Event: MWIGet
[ActionID:] <value>
Mailbox: <value>
OldMessages: <value>
NewMessages: <value>
Arguments
  • ActionID
  • Mailbox - Specific mailbox ID.
  • OldMessages - The number of old messages in the mailbox.
  • NewMessages - The number of new messages in the mailbox.
