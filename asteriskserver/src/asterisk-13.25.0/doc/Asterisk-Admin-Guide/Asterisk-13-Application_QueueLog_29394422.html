QueueLog()

Synopsis

Writes to the queue_log file.

Description

Allows you to write your own events into the queue log.

Example: QueueLog(101,${UNIQUEID},${AGENT},WENTONBREAK,600)

Syntax

QueueLog(queuename,uniqueid,agent,event,[additionalinfo])
Arguments
  • queuename
  • uniqueid
  • agent
  • event
  • additionalinfo
